# 前端脚手架

安装依赖
```bash
npm install
```

## 运行
```bash
# 本地运行
node ./bin/cli.js create

# 挂载到全局
npm link
f-cli create
```

## 安装
```bash
npm i @sunfengfeng/f-cli -g
```
